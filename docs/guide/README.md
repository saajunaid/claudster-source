# User guide

Team-facing user guide for the claudster + docket toolchain. These markdown files are the **canonical
source**; docket's in-app **Help** page renders copies of them (see `docket/web/src/help/`). Keep the two
in sync when editing.

- [start-here.md](start-here.md) — the map: the two tools, how they relate, and the three model lanes.
- [claudster.md](claudster.md) — the Claude Code harness: install, commands, memory model, every feature.
- [providers-and-keys.md](providers-and-keys.md) — GLM vs DeepSeek, where keys live, switching models, future-proofing.

The docket-specific guide lives with docket at `docket/web/src/help/docket.md`.
